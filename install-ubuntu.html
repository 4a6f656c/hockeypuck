Install on Ubuntu Server

Add the Hockeypuck project package repository

PPA prerequisites

In order to add PPA repositories, you'll need to make sure you have the necessary pacakges installed to manage PPAs on your server. These may already be installed on a desktop Ubuntu distribution, but may not be present on a minimal server image.

Ubuntu 12.04, 12.10

$ sudo apt-get install python-software-properties

Ubuntu 13.04

$ sudo apt-get install software-properties-common

Add the Hockeypuck repository

$ sudo apt-add-repository ppa:hockeypuck/ppa

Update the package list

$ sudo apt-get update

Installing Hockeypuck

For a standalone Hockeypuck server with its own database, install with the recommended packages:

$ sudo apt-get install hockeypuck

This will install postgresql, logrotate, and several other packages to install a ready-to-run, working service. If you do not want to install these packages, and want to configure the database and other services yourself, install without recommended packages:

$ sudo apt-get install --no-install-recommends hockeypuck
